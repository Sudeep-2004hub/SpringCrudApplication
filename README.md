# Student Management System

A Full Stack Student Management System developed using React.js for the frontend and Spring Boot for the backend. The application provides a user-friendly interface to perform CRUD (Create, Read, Update, Delete) operations on student records. It communicates through RESTful APIs and stores data in a MySQL database.

## Features
- Add new student records
- View all students
- Update student details
- Delete student records
- REST API integration
- Responsive user interface
- MySQL database connectivity

## Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Spring Boot, Spring Data JPA
- Database: MySQL
- Build Tool: Maven
- API Testing: Postman
